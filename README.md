**Global Death Rate Analysis (1960–2023) — Power BI Dashboard**

A complete analytical dashboard exploring global mortality trends across 60+ years using Power BI, DAX, and custom visualizations.

**Project Overview**
This project analyzes worldwide death rates from 1960 to 2023 using a fully interactive Power BI dashboard.
It highlights long-term mortality trends, year-over-year changes, country comparison insights, and predictive forecasting.
The dashboard uses:
1.Power Query for cleaning & transformation
2.DAX for custom metrics
3.Time-series analysis for trend insights
4.Shape Map with a custom TopoJSON world map
5.Forecasting visuals for future projections

**Dataset**
Source: Our World in Data (OWID)
Contains:
*Country
*Year
*Death rate
*Region
->Dataset cleaned and transformed using Power Query before analysis.

**Key Features**
**1. Descriptive Analysis**
Global death rate summary
Top & bottom countries
Regional distribution
High-level KPIs

**2. Temporal Trend Analysis**
Year-wise global and country trends
YOY % change
Regional improvements & declines
Moving averages

**3. Comparative Analysis**
Country vs Country comparison
Region-wise differences
Consistently high death-rate countries
Best and worst performing countries across decades

**4. Predictive Analysis**
Forecasting using Power BI analytics
Trendline with confidence intervals
Future projections up to 2030
